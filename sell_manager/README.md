# 介绍

前端：element-ui + vue + axios
后端：nodejs + mysql + express

1.登录功能模块，前后端验证。
2.接口鉴权token，后端采用jwt主流技术。
3前端axios分层封装，请求BASEAPI, timeout, 请求拦截器，响应拦截器。
4.动态面包屑。
5.路由动态管理，根据后端返回权限动态生成。
6.路由守卫实现用户界面权限控制。
7.账号模块增删改查分页，批量删除，修改密码，个人中心。
8.商品模块完整功能，图片上传。
9.店铺模块图片批量上传，完整交互。
10订单模块多条件高级模糊查询分页。
11.报表组件封装。
12.权限。


# 开始

前端
```
npm install
npm run serve
```
后端新建sell数据库安装nodemon依赖

```
nodemon
```
See [Configuration Reference](https://cli.vuejs.org/config/).
